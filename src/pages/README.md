# React Quiz App

Aplikasi kuis berbasis React dengan fitur:
- Login
- Soal dari OpenTriviaDB API
- Timer
- Satu soal per halaman
- Hasil kuis
- Resume kuis menggunakan localStorage

## Cara Menjalankan
1. Install dependencies
   npm install
2. Jalankan aplikasi
   npm start

## Teknologi
- React JS
- OpenTriviaDB API
- CSS
